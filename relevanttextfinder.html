Relevant Text Finder

Get all relevant text based on user input!
This project gets text like lemma(base form) of a verb, numbers in words, and synonyms relevant to the input along with their part of speech tags using python library SpaCy.
The motivation behind the project was to create an intelligent text searching app or extension that searches relevant text in documents based on the input.
