File Viewer

Browse files from a central location on Mac!
An app facilitating user to open files from a centralized place. Its Drag & Drop interface enables user to put several file paths in a single place and save them in a database file. The user can then access files by clicking or pressing enter at their corresponding path later from that centralized location.

This project was done for a client on freelancing website "Fiverr".
