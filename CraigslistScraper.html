Craigslist Scraper Desktop App

The above demo presents my desktop app that scrapes data from the famous craigslist website. Built with Python, it is a reliable & robust scraper that scrapes data based on certain parameters displayed in the application. After scraping, the user can also remove duplicates and store data in an excel sheet.

This project was done for a client on freelancing website "Fiverr".
